# .github
Если Roblox оказался заблокирован, UniVPN помогает восстановить доступ через стабильное подключение к безопасным серверам. Бот в Telegram выдаёт настройки автоматически, а промокод START открывает 3-дневный пробный период для проверки скорости и стабильности. Решение подходит для телефона и ПК.
